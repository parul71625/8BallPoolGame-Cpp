# 8BallPoolGame-Cpp

To learn about how to set OPENGL and other libraries, please refer to http://in2gpu.com/2014/10/15/setting-up-opengl-with-visual-studio/

The 8BallPoolGame.pdf gives a detailed introduction of the project. :)
